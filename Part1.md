## What is react.js -----
JS runs on  browser. Can manipulate DOM on page.

 ------> REQUEST (to server)

 <------ HTTP (from server)


* a client-side js library
* easy to build complex UI
* remove HTTP response from server

### React JS
easier to build complex internship
